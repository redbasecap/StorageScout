# **App Name**: RAKO Scout

## Core Features:

- Secure User Authentication: Enable users to log in securely using their Google account (OAuth) for personalized data management.
- QR Code Scan & Box Selection: Allow users to scan RAKO box QR codes using the device camera to automatically select or register a box UUID for item linking.
- Item Creation & Detail Input: Facilitate adding new items to a RAKO box, including its name/description, associated box UUID, and current physical location.
- Photo Capture & Attachment: Integrate device camera functionality to capture and attach one descriptive photo per item, stored securely.
- AI Item Description Suggestion Tool: Provide AI-powered suggestions for item names or descriptions based on the captured photo, to assist users with efficient data entry.
- Item Search & Location Lookup: Enable users to search for items by name or description and quickly view the corresponding box UUID and its physical storage location.
- PWA Core Functionality: Ensure the application is installable on user devices and provides essential offline capabilities for continued access to data.

## Style Guidelines:

- Primary color: A deep, professional blue-gray (#3E5572) to convey reliability and modern efficiency.
- Background color: A very light, almost white blue-gray (#EFF3F7) for a clean, minimalist canvas.
- Accent color: A bright, clear cyan (#51C2DA) for highlighting interactive elements and calls to action, providing good contrast.
- Body and headline font: 'Inter', a grotesque sans-serif font for its modern, objective, and highly readable characteristics, suitable for conveying clear information.
- Utilize a set of clean, outlined icons that are immediately recognizable and related to inventory management, scanning, and item details.
- Implement a clean, card-based layout for item display, prioritizing clear information hierarchy and intuitive navigation. Large, visible buttons for key actions like scanning and adding items.
- Incorporate subtle, functional animations for navigation transitions and providing visual feedback upon successful QR code scans or photo captures.
